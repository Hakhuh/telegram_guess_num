# Числовая угадайка — Telegram-бот

Telegram-бот на aiogram 3, реализующий игру «угадай число». Бот загадывает число в заданном диапазоне, а игрок пытается его отгадать, получая подсказки «больше» / «меньше». Работает как в личке, так и в группах.

![Демо работы бота](demo.gif)

## Возможности

- Игра «угадай число» с подсказками больше/меньше
- Настраиваемый диапазон загадывания через `/min` и `/max`
- Работа в личных чатах и в группах (в группе весь чат играет одним числом)
- Сохранение состояния между перезапусками (JSON)

## Команды

В личке:
- `/start` или `/again` — начать новую игру
- `/help` — справка
- `/min <число>` — задать минимальную границу диапазона
- `/max <число>` — задать максимальную границу диапазона
- Просто отправьте число — бот скажет больше оно или меньше загаданного

В группе все команды через `/ugadayka`:
- `/ugadayka start` — новая игра
- `/ugadayka help` — справка
- `/ugadayka min <число>` / `/ugadayka max <число>` — задать границы
- `/ugadayka <число>` — попытка угадать

## Установка и запуск

1. Клонировать репозиторий:
   ```
   git clone https://github.com/Hakhuh/telegram_guess_num.git
   cd telegram_guess_num
   ```

2. Создать и активировать виртуальное окружение:
   ```
   python -m venv .venv
   .venv\Scripts\activate      # Windows
   source .venv/bin/activate   # Linux/Mac
   ```

3. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```

4. Создать файл `.env` в корне проекта и вписать токен бота (получить у @BotFather):
   ```
   token=ВАШ_ТОКЕН_ЗДЕСЬ
   ```

5. Запустить:
   ```
   python main.py
   ```

## Стек

- Python 3.14
- aiogram 3

## Примечание

Для работы бота в группах нужно отключить privacy mode у @BotFather (`/setprivacy` → Disable), иначе бот не увидит обычные сообщения в группе.
